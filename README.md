# Celebrity Data Detective (Console Game)

## How to Run
1. Install Python 3.8+
2. Download the file 'Celebrity_detective_game.py'.
3. Open a terminal and run:

   python3 celebrity_detective_game.py

## How to Play
- Ask yes/no questions about the hidden object's features.
- When ready, type 'guess' and enter a label (0 or 1).
- The AI will also make a prediction using a decision tree.
- See who wins!
- Have fun!
